# Rbenv

📁 [Table of Contents](README.md)

## Description

Install versions of Ruby <https://github.com/rbenv/rbenv>

### Prerequisites

[Homebrew](homebrew.md)

### Steps

Install individually.

```sh
brew install rbenv
```

Also installs [ruby-build](ruby-build.md)

```sh
brew install rbenv
```

Configure System Path.

```sh
eval "$(rbenv init -)"
```



### Success Confirmation

```sh
which rbenv
```
