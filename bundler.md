## Description

[Ruby](ruby.md)

### Steps

```sh
gem install bundler
```

### Success Confirmation

```sh
bundle --version
```

```sh
bundle --version

```

## Usage

### Add Gem

Add the `fx` gem to a Bundler managed project

- Open `Gemfile`
- Add `gem 'fx'`
- From Terminal, run `bundle install`

`fx` should now be available.
